## Идея компании

Суть бизнеса: выйти на Ozon/Wildberries с определенной категорией товаров. Нужно проанализировать нишу, чтобы понять, какие бренды и размеры в дефиците, и какую цену ставить.

## Разделение обязанностей

- Романов Леонид - Конкуренты и ЦА: парсинг Wildberries/Ozon по запросу (цены, бренды, количество отзывов). А после поиск корреляции между ценой и количеством покупок (или отзывов). Сегментация ЦА по ценовым категориям (бюджет, мидл, премиум). 
- Погребной Евгений - Контрагенты и Логистика: парсинг оптовых китайских площадок (например, через каталоги 1688/Taobao). А после сравнение закупочной цены с ценой продажи и расчет маржинальности.
